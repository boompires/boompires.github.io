What is HTML?

HTML is the structure of your webpage kind of like the skeleton since our skeleton is the structure of our body. HTML is made up of tags. A tag is always surrounded by < and > which are the greater and less than signs you use in math. All the HTML in your webpage is wrapped in an html tag which looks like this: <html>. Now, what I just showed you is an opening tag which goes at the beginning of the content inside your tag. The closing tag is the same thing but goes after the content and has an extra slash which looks like this: </html>.

Another tag is the head tag. It looks like this: <head>. The closing tag looks like this: </head>. You usually put a title tag inside the head tag. The opening title tag looks like this: <title>. The closing title tag looks like this: </title>. Whatever text you put in the title tag is what appears in the tab at the top of your browser.

Another main tag is the body tag. All the content in your website goes inside the body tag. The opening body tag looks like this: <body>. The closing body tag looks like this: </body>. You put all the other tags inside the body tag (except the html and head tags) including the h1 tag, the p tag and the ul tag!

Here is a list of some basic HTML tags and what they do:

  • <h1> </h1> - this is a heading tag; they can go from h1 to h7 with h1 being largest and h7 being the smallest.
  • <p> </p> - this is a paragraph tag; Your main text content will usually go inside this tag.
  • <li> </li> - this is a list item tag; you use this tag to create items inside either an unordered list tag or an ordered list tag.
  • <ul> </ul> - this is an unorderd list tag; the items you put inside this tag is listed as bullets.
  • <ol> </ol> - this is an orderd list tag; the items you put in this tag are listed in a number list.


Here is an example of a basic HTML file. This code can be typed into a text editor and saved with a .html extension. And it will open in your browser. 

          
<html>
  <head>
    <title>Hi World!</title>
  </head>

  <body>
    <h1>Hi from my HTML page!</h1>
    <p>Hello, from West Hartford.</p>
  </body>
</html>
