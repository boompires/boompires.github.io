What is Javascript?

Javascript is the most popular programming language in the world. It's what makes HTML and CSS interactive by doing many useful things. With it you can make buttons, alerts, store information and process input from a form.

Javascript can be coded in your HTML page with script tags which look like this: <script> </script>. You can also code Javascript on a separate Javascript file which has to end with a .js extension.

Here is an example of some Javascript

      
<div id="listItem" onclick="addItem()">Click here to add an item 
to the list!</div>    

<script>
  function addItem() {
    let newItem = document.createElement('div');
    newItem.innerHTML = 'New Item!';
    document.getElementById('listItem').appendChild(newItem);
  }
</script>
