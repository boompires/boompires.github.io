What is CSS?

CSS means: Cascading Style Sheets

CSS is what makes your website not look bland. It's like the clothes we wear but for your website. CSS is always made up of a property and a value separated by a colon (:). First of all, CSS can be coded in your html inside a style tag that goes in your head tag. It looks like this: <style> </style>.

Another way to code CSS is on a separate CSS file. On the file you start with a selector (html element, class name, etc.). Then, you use curly braces which look like this: { }. In those braces you list properties and values separated by a colon (:).

Some common properties are:

  1. color
  2. background-color
  3. font-size
  4. text-align
  5. margin

Values can be given in words or numbers. Numbers can be in pixels, points or percentages.

An example of how you would write it in your code would be - margin: 20px; 

        
<style> 
  .styleDemo {
    width: 100px;
    height: 100px;
    background-color: red;
    display: inline-block;
    margin: 20px;
    font-size: 2em;
  }
</style>
